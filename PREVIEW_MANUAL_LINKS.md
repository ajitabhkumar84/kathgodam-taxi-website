# Manual Preview Links

Since the automatic preview is having issues with URL parameters, here are direct links you can use:

## Preview Links (Copy and Paste into Browser)

### Homepage
```
http://localhost:4321/api/preview?secret=preview-secret-kathgodam-taxi-2024&type=homePage
```

### Route Example (replace SLUG with actual route slug)
```
http://localhost:4321/api/preview?secret=preview-secret-kathgodam-taxi-2024&type=route&slug=SLUG
```

### Package Example (replace SLUG with actual package slug)
```
http://localhost:4321/api/preview?secret=preview-secret-kathgodam-taxi-2024&type=package&slug=SLUG
```

## How to Use

1. Copy the link above
2. Paste it in your browser
3. Press Enter
4. You should see the preview with yellow banner

## Exit Preview
```
http://localhost:4321/api/exit-preview
```
